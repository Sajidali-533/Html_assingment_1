1. strong tag

    • Use to indicate strong importance or emphasis.
    • Renders text in bold by default.
    • Use for semantic meaning rather than just for styling.

  2. u (underline tag)

    • Use to underline text.
    • However, underlining is generally discouraged for regular text as it can be confused with hyperlinks.
    • Reserve for specific cases like indicating misspelled words or stylistic choices in design elements.

  3. span tag

    • Use for styling inline elements within a block of text
    • Can apply CSS styles or classes to specific parts of the text.
    • Avoid using for formatting whole paragraphs; use for smaller, inline elements instead.
